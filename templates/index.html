How to Use This App

  • Add a task: Type in the text box and press Enter or click "Add Task"
  • Complete a task: Click the checkbox to mark it done
  • Undo completion: Click the checkbox again to unmark it
  • Delete a task: Click the red Delete button (you'll be asked to confirm)
  • Set a reminder: Choose a task, pick a date/time, and click Set Reminder
  • Check weather: Enter a city name in the weather widget
  • Export tasks: Click Export to JSON or Export to Text at the bottom
